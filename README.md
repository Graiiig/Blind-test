# blind-test-cli

Un projet de Blind Test en vue js 3 couplé à du Firebase et Spotify.  

## Installation

Remplir le .env avec les infos Firebase, Spotify et Unsplash.

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
